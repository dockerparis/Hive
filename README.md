# README #


### Summary ###

* Django Open Calcul Tool
* Version: 0.1


### How do I get set up? ###

* $> virtualenv env
* source env/bin/activate
* pip install -r requirements.txt
* $> python manage.py syncdb
* $> python manage.py loaddata doct/app/migrations/fixtures/tasks.json
* $> python manage.py runserver
